# **App Name**: Web Data Forge

## Core Features:

- Hero Section: Impactful title combining web and data, client-pain-addressing subtitle, clear CTA button.
- Value Proposition Cards: Three cards explaining the unique approach to web development, data intelligence, and integrated strategy.
- Featured Services: Clear blocks offering landing page creation, web applications (SaaS), data analysis and reports, and AI integration.
- Project Showcase: Grid layout displaying previous works with title, technologies used (tags), and live link.
- About Me Section: Brief bio mentioning academic background (Data Science) and freelance experience to add a personal touch.
- Footer Links: Links to social networks (LinkedIn, GitHub) and contact form or WhatsApp button.
- Animated Data Flow: Subtle visual effect using animations or gradients to suggest a 'data flow'.

## Style Guidelines:

- Color scheme: Dark mode to evoke technology and data. Background color: Dark gray (#121212).
- Primary color: Neon cyan (#00FFFF) to add technological accent. This will contrast nicely with the dark background.
- Accent color: Analogous color to the primary; blue-violet (#4F35FF) to complement and provide visual interest.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text to achieve a modern and technical look.
- Lucide-react or heroicons to provide minimalistic and consistent icons throughout the landing page.
- Fully responsive layout optimized for various screen sizes using Tailwind CSS's responsive utilities.
- Subtle data flow animations or gradients to provide a modern, dynamic feel, suggesting movement and processing.
